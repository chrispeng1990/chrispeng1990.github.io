    1. 创建一个资源库专用Repository #

    2. 生成免密token #

    点击用户头像 -> Settings -> Developers Settings;
    创建新token: Personal access tokens -> Tokens -> Generate New Token

    过期时间选择永不过期; 权限范围勾选repo所有权限.

    创建生成token, 注意保存token, 关闭页面后无法再查询.
