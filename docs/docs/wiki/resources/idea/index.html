    1. 破解 #

    1.1. Agent方式 #

    jetbrains-agent.jar

    试用版本 2019.2 及以下

    1. 打开IDEA -> Help -> Edit Custom VM Options
    2. 首次需要创建文件, 并且按照模版填写内容
    3. 并在文件最后追加内容:
    -javaagent:/absolute/path/to/jetbrains-agent.jar
    1. 重启 IDEA
    2. Help -> Register
      两种注册方式:
    • License server方式
      地址填入: http://jetbrains-license-server （应该会自动填上）
    • Activation code方式离线激活
      使用如下激活码, 如果激活窗口一直弹出(error 1653219), 在hosts文件里移除jetbrains相关的项目, 如果需要自定义License name, 参考: https://zhile.io/custom-license.html
