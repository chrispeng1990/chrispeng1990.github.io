    1. 安装node版本管理器 - nvm #

    1.1. 下载nvm #

    Github/nvm

    1.2. 配置nvm源 # 

    nvm node_mirror https://npm.taobao.org/mirrors/node/
nvm npm_mirror https://npm.taobao.org/mirrors/npm/

    2. 安装nodejs #

    2.1. 安装node # 

    # 查看远程库可用版本列表
nvm list available
# 安装指定版本
nvm instsall 8.11.2
# 查看本地版本列表
nvm ls

    2.2. 使用指定版本 # 

    nvm use 8.11.2

    2.3. 设置默认版本 # 

    nvm alias default 8.11.2
nvm ls

    2.4. 设置版本别名 # 

    nvm alias nickname 8.11.2
nvm use nickname

    2.5. 取消别名 # 

    nvm unalias nickname

    3. 安装npm镜像管理器 - nrm #

    3.1 安装nrm # 

    npm install -g nrm

    3.2. 修改镜像源 # 

    nrm ls
nrm use taobao
