## 1. 国内域名
域名提供商: 阿里云、腾讯云、花生壳  
限制:  
+ 需要备案才能使用域名, 可在小程序上备案;
+ 必须要有 3 个租期(月)以上的服务器才能开始备案，而且一台服务器只能为一个域名备案。
  

## 2. 国外域名
域名服务商: [NameSilo](https://www.namesilo.com/) 和 [GoDaddy](https://www.godaddy.com/zh-sg)  
修改CNAME、A记录后24小时生效  

## 3. 其他说明
1. 国内的域名是不能用在国外服务器的, 所以代理Github只能购买国外域名  
2. 域名可以更换服务商, 也支持国内国外互转, 但转完 3 个月内域名是被冻结的状态, 不能使用
